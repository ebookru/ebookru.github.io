# Книги и статьи

**Smart Things Bookstore** — готовые базы знаний для тех, кто хочет научиться работать с ИИ (ищет знания и желает получить реальный старт в освоении агентов), с криптовалютой и трейдингом (знания, которые пережили много рыночных циклов — практические знания и советы из книг авторов, которые подтвердили свои методы деньгами, плюс системный подход) и для программистов (начинающих или профи, кто учится или ищет знания без перелопачивания всего интернета). Мы уже всё приготовили — берите и пользуйтесь!

Собранные, переведённые и каталогизированные архивы для Obsidian: разверни готовый Second Brain и получи структурированные знания вместо потери времени на поиск информации по разрозненным источникам

Каждая статья переведена на русский язык с сохранением разметки; осмысленным текстом (там где это возможно) и общепринятой терминологией. Материал сортирован по тематическим папкам без дублирования статей. Всё что вам нужно уже там, в том числе монетизация и создание бизнеса с помощью ИИ; теперь ничего не потеряется и вы легко сможете добавить своё по шаблону.

Все изображения сконвертированы в едином современном формате AVIF, а архив теперь занимает намного меньше места, картинки в статьях открываются мгновенно. Каждая заметка оформлена в Markdown и имеет заполненные свойства (properties), поэтому Obsidian сразу понимает, как связывать материалы между собой. Связи между статьями настроены так, чтобы граф знаний (Graph View) сразу показывал реальную картину — не пустое поле, а рабочую карту знаний.

> [!tip]  «Скормите» статьи вашему агенту, или «предложите» ему для ответов использовать эту базу.
